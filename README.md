# Meridian

Meridian ist ein ressourcenschonender Wayland-Compositor in Rust mit separatem Shell-Prozess (`meridian-shell`).

## Workspace
- `meridian-compositor`
- `meridian-shell`
- `meridian-config`
- `meridian-ipc`
- `meridian-wm`
- `meridian-portal` (Scaffold)

## Schnellstart
- Build: `cargo build`
- Run: `cargo run`
- Tests: `cargo test --workspace`
- Check: `cargo check --workspace`
